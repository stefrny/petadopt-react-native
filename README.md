# Pet Adopt 🐾

Aplicativo de adoção de pets desenvolvido com Expo e React Native.

## 🎯 Objetivos Principais

- [ ] **Autenticação Real:** Implementar login e cadastro integrados à API e persistência de sessão.
- [ ] **Rotas Protegidas:** Garantir que apenas usuários logados acessem o app principal.
- [ ] **Listagem de Pets com Filtros:** Melhorar a Home com categorias e filtros rápidos.
- [ ] **Filtros Avançados:** Modal para busca refinada (espécie, porte, idade, etc).
- [ ] **Busca de Pets:** Barra de pesquisa por nome ou localização.
- [ ] **Detalhes do Pet:** Tela completa com informações detalhadas e contato.
- [ ] **Favoritos:** Funcionalidade de salvar pets favoritos localmente.
- [ ] **Perfil do Usuário:** Tela de visualização de dados e logout.
- [ ] **Navegação por Tabs:** Reestruturar abas (Home, Favoritos, Perfil).
- [ ] **UI/UX Update:** Modernizar a interface com cores consistentes, animações e feedback visual.

## 🛠️ Tecnologias

- React Native / Expo
- Expo Router
- Context API
- AsyncStorage
- Fetch API

## 🚀 Como rodar o projeto

1. Instale as dependências: `npm install`
2. Inicie o projeto: `npx expo start`
